# Writing Samples — Brain Food 학습 데이터

HS의 과거 글쓰기 샘플. Brain Food 에이전트가 문체·스타일 학습에 사용.

## 구조

```
writing_samples/
├── linkedin/     # LinkedIn 포스트 (.md)
├── telegram/     # Telegram 채널 포스트 (.md, 자동 저장)
└── README.md
```

## 자동 저장
Brain Food 텔레그램 채널에 글을 올리면 `telegram/`에 자동 저장됩니다.

## 수동 추가
LinkedIn 포스트는 `linkedin/YYYY-MM-DD-제목.md` 형식으로 저장하세요.

## 파일 형식

```markdown
# 포스트 제목 (or 첫 줄)

날짜: 2026-03-12
플랫폼: LinkedIn

---

(포스트 본문)
```
